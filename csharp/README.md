# Refactorizacion Gilded Rose

Refactorizacion

- Uso de Forech en lugar de for
- Creación de métodos privados para extraer codigo repetido
- Creación de variables estaticas para reemplazar cadenas repetidas
- Reducir las estructuras anidadas If/else
- Uso de Polimorfismo
- Creacion de clases para separar el comportamiento de actualizacion de estado

Uso Patrón

- Uso del patron Estrategia, con el objetivo de recorrer cada elemento y aplicar la función de actualización a cada elemento independientemente de saber  el tipo de cada elemento.

- Uso del patron fabrica para crear y devolver la instancia de estrategia para el tipo unico, resolviendo el problema de mapeo de encontrar la estrategia correcta.
