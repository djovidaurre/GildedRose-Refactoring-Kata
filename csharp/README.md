# Refactorizacion Gilded Rose

Procedimientos de Refactorizacion

- Usar Forech en lugar de for
- Creacion de métodos privados para extraer codigo repetido
- Creacion de variables estaticas
- Reducir las estructuras anidadas If/else
- Uso de Polimorfismo
- Creacion de clase para cada tipo de elemento

Uso Patrón

- Uso del patron Estrategia, con el objetivo de recorrer cada elemento y aplicar la funcion de actualizacio a cada elemento independientemente de saber  el tipo de cada elemento.

- Uso del patron fabrica para crear y devolver la instancia de estrategia para el tipo unico, resolviendo el problema de mapeo de encontrar la estrategia correcta.
